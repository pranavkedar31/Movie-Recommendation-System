# 🎬 Movie Recommendation System

A Movie Recommendation System built using Streamlit, Machine Learning, and OMDb API.

## Features

- Movie Search
- Search-as-you-type
- Movie Details
- IMDb Ratings
- Posters
- Trailer
- Similar Movie Recommendations
- Wikipedia Link
- Watch Online Link
